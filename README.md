# Vibez Hookah Bar Lounge – Website

Mehrseitige Website für die Vibez Hookah Bar Lounge in Kriftel.

## Dateien

| Datei | Inhalt |
|---|---|
| `index.html` | Startseite |
| `menu.html` | Speisekarte (Beispiel!) |
| `gallery.html` | Galerie |
| `kontakt.html` | Kontakt & Reservierung |
| `impressum.html` | Impressum |
| `datenschutz.html` | Datenschutzerklärung |
| `style.css` | Gemeinsames CSS für alle Seiten |

## Hinweise

- **Speisekarte**: Die Speisekarte ist eine **Beispielkarte** und muss mit echten Preisen/Produkten ersetzt werden.
- **Impressum**: Bitte Inhabername/Vertretungsberechtigten ergänzen und rechtlich prüfen lassen.
- **Datenschutz**: Bitte vor Live-Gang von einem Datenschutzanwalt prüfen lassen.
- **Reservierungsformular**: Momentan mit Alert-Platzhalter – ggf. an ein Backend/Formular-Service anbinden.
- **Google Fonts**: Werden von Google geladen – ggf. lokal hosten für besseren Datenschutz.

## Lokales Testen

Einfach `index.html` im Browser öffnen – alle Seiten sind relativ verlinkt.

## GitHub Pages

1. Repository erstellen
2. Dateien hochladen
3. In Settings → Pages → Branch `main` / Root auswählen
4. Website ist unter `https://USERNAME.github.io/REPO/` erreichbar
